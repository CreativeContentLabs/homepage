    연구목표

    구분 연구개발 목표 연구개발 내용 및 범위
    1차년도 영상물 워터마킹기술
    • 영상물을 위한 공격에 안전한 워터마킹 기술 개발
    장소 매핑 영상물 콘텐츠 저작권 참조 모델
    • 공동저작물과 결합저작물 측면에서 영상물 콘텐츠 저작권 권리 분류 방법 개발
    • 각각의 저작물을 구분할 수 있는 3가지 측면의 저작권 권리 분류 방법
    장소 매핑 영상물 콘텐츠 과금 모델
    • 저작권 참조 모델에 의해서 분류된 콘텐츠를 적용할 수 있는 마이크로 라이선싱을 이용한 과금 모델 개발 및 적용
    2차년도 개선된 영상워터마킹기술
    • 삽입 정보량을 증가하고 공격에 안전한 영상물 워터마킹 기술 개발
    장소 매핑 영상물 콘텐츠 저작권 참조 모델
    • 장소 매핑 영상물 콘텐츠를 구분할 수 있는 5가지 측면의 저작권 권리 분류 방법 개발
    장소 매핑 영상물 콘텐츠 과금 모델
    • 저작권 참조 모델에 의해서 분류된 콘텐츠를 적용할 수 있는 마이크로 라이선싱을 이용한 개선 된 과금 모델 개발 및 적용
    3차년도 영상 워터마킹기술 library화
    • 특징 기반 이미지/비디오 워터마킹 최적화
      • 워터마킹 알고리즘을 최적화 하여 삽입 정보량을 늘이되 비지각성에는 큰 영향이 없도록 개선
    • 개발기술의 라이브러리구축
      • 상용화사용을 위하여 라이브러리를 구축
    장소 매핑 영상물 콘텐츠 저작권/과금 상용화 모델 상용화 검증
    • 영상물 콘텐츠를 5가지 저작권 참조 모델로 분류 후 마이크로 라이선싱을 적용한 과금 모델에 적용.
    • 워터마킹 라이브러리를 이용.
